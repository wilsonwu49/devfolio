WiFi-Controlled RC Car with Speaker

Wi-Fi controlled robotic car built using an STM32L432KC and ESP32 with a Python client interface. The system separates networking and real-time control across two microcontrollers for responsive motor control and audio output while handling wireless communication.

  • Dual-microcontroller architecture: ESP32 handles Wi-Fi communication while STM32 performs real-time motor and audio control
  • Python client sends commands over a local Wi-Fi network hosted by the ESP32
  • Low-latency I²C communication between ESP32 and STM32 for command transfer
  • STM32 PWM motor control driving two DC motors through an H-bridge for speed and turning control
  • Sounds generated using STM32 DMA-driven DAC output connected to an amplifier and speaker

Snake Arcade Game (VHDL)

Final project for ES4: Digital Logic Circuits. A complete implementation of the arcade game Snake written in VHDL and deployed on an ICEUP5K (iCE40 UltraPlus) FPGA.

  • Snake game implemented entirely in VHDL on an FPGA, rendered over VGA (640×480) on a 16×16 grid
  • NES gamepad input with real-time score and high-score tracking
  • Custom pixel-art graphics for the snake, apple, start screen, and game-over screen
  • Finite state machine controlling movement, wall/self collision detection, tail updates, and RNG apple spawning
  • RAM game board encoding: snake body, apple, and background
  • Stable game tick and VGA timing derived from 12 MHz system clock

  • Implemented game sprites, graphics, and pattern generation for VGA display
  • Modified ROM address manipulation to support grid manipulation
